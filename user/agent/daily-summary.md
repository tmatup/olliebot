# Daily Summary Task

Use GitHub MCP to check UiPath\Jarvis project outstanding PR and recent commits

## The work

Run this task once a day at 7AM on the past 24 hours

* Summarize the commits in the past 24 hours by reading the info in associated pull requests. Create a short description for each of the commits based on its pull request description. Do not just use the raw pull request description that is usually too noisy.
* Summarize a list of outstanding Pull Requests

## Output

- A report nicely formatted in markdown. Executive summary on top commmeting on the general area being worked on based on both commit and also on going pull requests. Follow by a bullet list of commits in the past day, and then a bullet list of on going pull requests.
